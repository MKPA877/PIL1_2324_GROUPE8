IV.Déploiement de l'application

Veuillez suivre les instructions suivant pour un déploiement de l'application

-Installation de Django: 

                pip install Django

-Installation de Channels et daphne: 

                python -m pip install -U channels["daphne"]

-Création d'un environnement virtuel dans le répertoire du projet: 

                python -m venv .venv

-Installation des outils pour une connection avec postgre Sql: 

                pip install psycopg2-binary

-Installation de postgre Sql et configuration des paramètres de connection:

Créer une BD avec nom : "project"

Configurer le port avec le port de votre Postgre Sql : "PORT":"default"

Mettre à jour vos paramètre de connection à la base de donnée defini par defaut sur USER et PASSWORD

                
                    DATABASES = {
                        'default': {
                            'ENGINE': 'django.db.backends.postgresql',
                            'NAME': 'project',
                            'USER': 'user',
                            'PASSWORD': 'password',
                            'HOST': 'localhost',
                            'PORT': '5432',
                        }
                    }

A defaut on peut installer les dépendances python requises pour le projet avec la commande: 

                pip install -r requirements.txt

Cette commande permet d'installer tous les modules pythons dont un projet a besoin en une seule fois

-Executer les migrations : 

                 python manage.py migrate

-Lancer le server de developpement

                python manage.py runserver

-Accédez a l'application dans votre navigateur en utilisant l'URL:

                http://127.0.0.1:8000/.

-Créer un superutilisateur

                python manage.py createsuperuser

-Se connecter a l'interface d'administration django par l'URL:

                http://127.0.0.1:8000/admin/.

Ainsi l'application sera deployée. Merci pour votre attention.
